# Next.js 시작하기: 웹 개발의 새로운 시대

Next.js는 React 기반의 풀스택 웹 프레임워크로, 현대적인 웹 애플리케이션을 쉽게 구축할 수 있게 해줍니다.
서버 사이드 렌더링(SSR), 정적 사이트 생성(SSG), API 라우트 등 다양한 기능을 제공합니다.

## Next.js의 주요 기능

### 1. 하이브리드 렌더링

Next.js는 페이지별로 렌더링 방식을 선택할 수 있는 유연성을 제공합니다:

- **서버 사이드 렌더링(SSR)**: 페이지 요청 시마다 서버에서 HTML을 생성
- **정적 사이트 생성(SSG)**: 빌드 시점에 HTML을 생성하여 CDN에서 제공
- **증분 정적 재생성(ISR)**: 특정 간격으로 페이지를 재생성

### 2. 자동 코드 분할

각 페이지는 필요한 JavaScript만 로드하기 때문에 초기 로딩 시간이 빠릅니다.

```jsx
// 페이지 컴포넌트 예시
export default function HomePage() {
  return (
    <div>
      <h1>Next.js 시작하기</h1>
      <p>웹 개발의 새로운 시대에 오신 것을 환영합니다!</p>
    </div>
  );
}
```

### 3. 파일 기반 라우팅

pages 디렉토리(App Router의 경우 app 디렉토리)에 파일을 생성하면 자동으로 라우트가 구성됩니다.

## Next.js 시작하기

프로젝트 생성은 다음과 같이 간단합니다:

```bash
npx create-next-app@latest my-next-app
cd my-next-app
npm run dev
```
